# Part 2: NoSQL Database Analysis (MongoDB)

## Objective
This part evaluates the suitability of MongoDB for handling a flexible product catalog with diverse attributes and nested data.

## Contents
- `nosql_analysis.md`: Theoretical comparison of RDBMS vs MongoDB, including benefits and trade-offs.
- `mongodb_operations.js`: MongoDB queries and updates demonstrating CRUD and aggregation operations.
- `products_catalog.json`: Sample product catalog data used for MongoDB operations.

## Key Features
- Flexible document schema
- Embedded review documents
- Aggregation pipelines for analytics
